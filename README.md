## Features

- **Dark neon color scheme** optimized for MongoDB, Express, React, and Node.js development
- **Syntax highlighting** with neon colors for better code readability
- **File icons** specifically designed for MERN stack files
- **Semantic highlighting** for enhanced code understanding

## Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X)
3. Search for "MERN Neon Theme"
4. Click Install

## Usage

### Color Theme

1. Open Command Palette (Ctrl+Shift+P)
2. Type "Preferences: Color Theme"
3. Select "MERN Neon Flow"

### File Icon Theme

1. Open Command Palette (Ctrl+Shift+P)
2. Type "Preferences: File Icon Theme"
3. Select "MERN Neon Icons"

## Color Palette

- **Keywords**: Red (#ff6b6b)
- **Strings**: Yellow (#f6c177)
- **Variables**: Green (#4ade80)
- **Functions**: Cyan (#22d3ee)
- **Numbers**: Pink (#f472b6)
- **Classes**: Purple (#a78bfa)
- **Comments**: Green (#6a9955)

## Supported File Types

- JavaScript (.js)
- TypeScript (.ts)
- React (.jsx, .tsx)
- JSON (.json)
- Environment files (.env)
- MongoDB files (mongoose.js)

## Development

To test locally:

1. Clone this repository
2. Open in VS Code
3. Press F5 to launch a new Extension Development Host window
4. Test the theme and icons
